Java - Assisted learning

Being one of the most versitile languages in the world, Java runs on 3 billion devices. Java helps introduce the concepts of object oriented programming to beginner programmers and is one of the most prominent programming language used.
This program was designed for my ICS4U computer science class, the purpose of this program is to help low level math students improve their arithmetic equation solving skills. This program produces randomly generated addition, subtraction and multiplication equations.
While the user is completing questions the program will automatically produce randomly selected feedback that provide positive reinforcement. Whether the user inputs the correct answer or incorrect answer will differ the response recieved.
At the end of the program the system will output your overall score as well as display the amount of questions you got right with the amount of attempts used. The score is presented in a percentage using the format print function in java. Above are more examples of randomly selected positive responses after each inputted answer.

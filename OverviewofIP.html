Overview of IP

IP stands for internet protocol, IP addresses on the other hand are made from bunch of numbers assigned to devices connected in a network used to communicate through internet protocol.

These numbers allow for identification as well as locating devices for both hosts and users, similarly to house addresses IP addresses help allow information or data packets to arrive from the host to the requested user without sending it to anyone else. Internet protocol also help predict the most efficient and quick way to deliver data packets to allow for the fastest path possible. This makes an efficient and private way to surf the web.
